# 👨‍💼 AI Employee Attrition Prediction Platform

An AI-powered Human Resource Analytics application that predicts whether an employee is likely to leave an organization using Machine Learning.

---

## 📌 Project Overview
Employee attrition is one of the biggest challenges faced by organizations. Losing experienced employees leads to increased recruitment costs, lower productivity, and knowledge loss. 

This project leverages Machine Learning (built with **Python**, **Flask**, and **Scikit-Learn**) to analyze employee demographics, compensation, and work environment factors to predict attrition risks in real time.

---

## 🛠️ Tech Stack
* **Programming Language:** Python
* **Web Framework:** Flask
* **Machine Learning:** Scikit-Learn, NumPy, Pandas
* **Frontend:** HTML5, CSS3
